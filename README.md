## Typescript Client library for mythtv services API
### Supports
* Backend APIs
  * Channel Service
    * GetChannelInfo
    * GetVideoSourceList
    * GetChannelInfoList
  * DVR Service
    * GetEncoderList
    * GetRecordedList
    * GetUpcomingList
  * Myth Service
    * GetHostName
    * GetHosts
    * GetSetting
  * Video Service
    * GetVideoList
* Frontend APIs
  * GetActionList
  * GetStatus
  * SendMessage
  * SendAction
  * SendKey
  * PlayRecording
  * PlayVideo
  * GetContextList
  * SendNotification
